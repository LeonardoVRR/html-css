Lista de Definições

Para criar um lista de definições basta usar a tag abaixo:

        
            <dl>
                <dt>Termo</dt>
                <dd>Descrição</dd>
            </dl>

Exemplo de uso:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
        
            <dl>
                <dt>HTML</dt> (Termo)
                <dd>Linguagem de marcação para a criação do conteúdo de um site.</dd> (Descrição)
            </dl>
