Trabalhando com tabelas

para colocar uma tabela basta a tag <table></table>

para adicionar linhas usa a tag <tr><tr>

para adicionar as colunas usa a tag <td><td>

Exemplo de codigo

        
            <table>
                <tr>
                    <td>
                        item dentro da tabela
                    <td>
                <tr>
            </table>

Exemplo

linha 1 coluna 1; linha 1 coluna 2; linha 1 coluna 3;
linha 2 coluna 1; linha 2 coluna 2; linha 2 coluna 3;
linha 3 coluna 1; linha 3 coluna 2; linha 3 coluna 3;
